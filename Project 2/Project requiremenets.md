ECE9124 - Ανάπτυξη Εφαρμογών Παγκόσμιου Ιστού
Εργασία 2 Tables + CSS Layout + Effects
Θέμα: Κατασκευάστε έναν ιστότοπο
Περίληψη
Ο στόχος σας σε αυτή την εργασία είναι κατασκευάσετε έναν ιστότοπο (web site) αποτελούμενο από δύο
ιστοσελίδες με θεματολογία της επιλογής σας. Μπορείτε να τροποποιήσετε/επεκτείνετε την ιστοσελίδα που
αναπτύξατε για τις ανάγκες της 1ης εργασίας.
Οδηγίες
Μπορείτε να χρησιμοποιήσετε κώδικα από τα παραδείγματα/ασκήσεις που πραγματοποιήσαμε στην
διάρκεια των μαθημάτων, αλλά θα πρέπει να τον προσαρμόσετε-επεκτείνετε κατάλληλα.
Δεν πρέπει να χρησιμοποιήσετε εργαλεία συγγραφής κώδικα τύπου WYSIWYG (π.χ. Dreamweaver).
Θα παραδώσετε την εργασία μέσω του eclass, σε αρχείο .zip, που θα έχει την μορφή epithetο_onoma.zip
(π.χ. fevgas_athanasios.zip). Το αρχείο .zip θα περιέχει και την τεκμηρίωση (project02_epitheto.pdf).
Απαιτήσεις
§ Ο ιστότοπος που θα παραδώσετε θα αποτελείται από δύο αρχεία .html, ένα ή περισσότερα αρχεία .css,
τυχόν εικόνες, και γραφιστικά στοιχεία.
§ Το 1ο HTML έγγραφο θα αποτελεί την κεντρική σελίδα του ιστοτόπου (home page), ενώ το 2ο θα είναι
μια φόρμα επικοινωνίας.
§ Η φόρμα επικοινωνίας θα πρέπει να διαθέτει τουλάχιστον 5 στοιχεία φόρμας της επιλογής σας.
Εφαρμόστε στυλ στα στοιχεία της φόρμας.
§ Θα πρέπει να χρησιμοποιήσετε έναν τουλάχιστον πίνακα (HTML). Tο πλήθος των στοιχείων που θα
περιέχει είναι της επιλογής σας.
§ Πρέπει να χρησιμοποιήσετε CSS για τη μορφοποίηση του περιεχομένου και τη διάταξη. Χρησιμοποιείστε
διάταξη πλέγματος (Grid Layout) ή/και Flexbox.
§ Θα πρέπει να επιδείξετε την χρήση ενός τουλάχιστον οπτικού εφέ με CSS.
§ Η υλοποίηση σας θα πρέπει να ακολουθεί τις αρχές της ανταποκρινόμενης σχεδίασης (responsive
design). Να περιλαμβάνει τουλάχιστον 2 media queries.
§ Η υλοποίησή σας θα πρέπει να ακολουθεί τους κανόνες προσβασιμότητας.
Τεκμηρίωση
Η εργασία σας θα πρέπει να συνοδεύεται από κατάλληλη τεκμηρίωση. Η τεκμηρίωση θα πρέπει να
παραπέμπει στην υλοποίηση των παραπάνω απαιτήσεων στα αρχεία .html και .css.
