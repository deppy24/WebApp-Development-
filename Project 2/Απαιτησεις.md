1.Ο ιστότοπος που θα παραδώσετε θα αποτελείται από δύο αρχεία .html, ένα ή περισσότερα αρχεία .css,
τυχόν εικόνες, και γραφιστικά στοιχεία
        index2.html,index22.html,style2.css,formstyle.css
2.Το 1ο HTML έγγραφο θα αποτελεί την κεντρική σελίδα του ιστοτόπου (home page), ενώ το 2ο θα είναι
μια φόρμα επικοινωνίας. 
         <li><a href="/Project 2/index22.html">Contact Form</a></li> <!--H προσθηκη επιλογης στο nav bar για συνδεση με την νεα σελιδα-->
3.Η φόρμα επικοινωνίας θα πρέπει να διαθέτει τουλάχιστον 5 στοιχεία φόρμας της επιλογής σας.
Εφαρμόστε στυλ στα στοιχεία της φόρμας.
        <input type="text" placeholder="Your Name" required>
        <input type="text" placeholder="Your Surname" required>
        <input type="email" placeholder="Your Email" required>
        <input type="tel" placeholder="Your Phone number" required>
        <textarea placeholder="Your Message" required></textarea>

        form label {
            margin-top: 10px;
            font-weight: bold;
            font-size: 18px;
            }

5.Πρέπει να χρησιμοποιήσετε CSS για τη μορφοποίηση του περιεχομένου και τη διάταξη. Χρησιμοποιείστε
διάταξη πλέγματος (Grid Layout) ή/και Flexbox.
        div form {
            display: flex;
            flex-direction: column;
            width: 400px;
            gap: 10px;  }
6.Θα πρέπει να επιδείξετε την χρήση ενός τουλάχιστον οπτικού εφέ με CSS
        form button:hover {
        background-color: #a4511d; /* Αλλαγή χρώματος φόντου */
        transform: scale(1.1); /* Μεγέθυνση του κουμπιού */
        box-shadow: 0 4px 10px rgba(0, 0, 0, 0.2); /* Προσθήκη σκιάς για βάθος */
            }
